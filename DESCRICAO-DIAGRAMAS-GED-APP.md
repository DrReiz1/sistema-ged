**Diagrama de Atividade**
Este diagrama representa o fluxo operacional do uso do aplicativo integrado ao GED. Ele demonstra a sequencia de acoes desde a aproximacao do cracha NFC pelo operador, passando pela validacao de acesso, consulta dos documentos autorizados e abertura do arquivo, ate o registro final da acao no historico do sistema. Seu objetivo e mostrar o comportamento geral do processo de uso.

**Diagrama de Sequencia**
Este diagrama apresenta a ordem temporal das mensagens trocadas entre os principais elementos do projeto: operador, aplicativo, GED e Supabase. Ele evidencia como ocorre a autenticacao por NFC, a consulta dos documentos permitidos e o registro de logs, destacando a sequencia de interacoes necessaria para que o fluxo funcione corretamente.

**Diagrama de Comunicacao**
Este diagrama descreve como os componentes do sistema se relacionam entre si durante a operacao. O foco nao esta no tempo de execucao, mas na troca de informacoes entre operador, aplicativo, GED API e Supabase. Ele permite visualizar de forma objetiva a arquitetura de comunicacao adotada na integracao entre o sistema web e o aplicativo.
